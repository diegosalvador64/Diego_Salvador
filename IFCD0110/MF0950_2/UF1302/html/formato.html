Formato de HTML

El formato de texto es el marcado que se aplica a un texto para añadirle datos de forma

Al texto resultante se le conoce como texto formateado, texto con estilos o texto enriquecido

Encabezados de texto

Los motores de búsqueda utilizan los encabezados para indexar la estructura y el contenido de sus páginas web.

Los usuarios hojean sus páginas por sus títulos. Es importante utilizar los encabezados para mostrar la estructura del documento.

Los encabezamientos <h1> deben ser utilizados para los encabezados principales, seguido por encabezados <h2>, y menos importantes sucesivamente: <h3>, <h4>...

Nota: Usar encabezados HTML para sólo encabezados. No utilice encabezados para que el texto GRANDE o negrita.

Etiqueta Descripción Ejemplo Utilización
<h1> Describe un encabezado de tipo 1

Encabezado tipo 1

 <h1>Encabezado tipo 1</h1>
<h2> Describe un encabezado de tipo 2

Encabezado tipo 2

 <h2>Encabezado tipo 2</h2>
<h3> Describe un encabezado de tipo 3

Encabezado tipo 3

 <h3>Encabezado tipo 3</h3>
<h4> Describe un encabezado de tipo 4

Encabezado tipo 4

 <h4>Encabezado tipo 4</h4>
<h5> Describe un encabezado de tipo 5
Encabezado tipo 5
 <h5>Encabezado tipo 5</h5>

Formateo de elementos HTML

HTML también define etiquetas especiales para la definición del texto con un especial significado.

HTML utiliza etiquetas para el formato de salida, como negrita o cursiva de texto.

Etiqueta Descripción Ejemplo Utilización
<b> Define un texto en negrita Este texto está en negrita <b>Este texto está en negrita</b>
<i> Define un texto en cursiva Este texto está en cursiva <i>Este texto está en cursiva</i>
<strong> Define un texto importante Este texto es importante <strong>Este texto está importante</strong>
<em> Define un texto enfatizado Este texto está enfatizado <em>Este texto está enfatizado</em>
<small> Define un texto más pequeño Este texto es más pequeño Este texto es <small>más pequeño</small>
<sup> Define un texto superíndice Este texto es superíndice de texto Este texto es <sup>superíndice</sup> de texto
<sub> Define un texto subíndice Este texto es subíndice de texto Este texto es <sub>subíndice</sub> de texto
<ins> Define un texto insertado o agregado Este texto es insertado o agregado Este texto es <ins>insertado o agregado</ins>
<s> Define un texto tachado Este texto es tachado <s>Este texto es tachado</s>
<u> Define un texto subrayado Este texto es subrayado <u>Este texto es subrayado</u>
<del> Define un texto borrado o eliminado Este texto es borrado Este texto es <del>borrado</del>
<mark> Define un texto marcado Este texto es marcado Este texto es <mark>marcado</mark>

Nota: Los navegadores muestran <strong> como <b>, y <em> como <i>. Sin embargo, hay una diferencia en el significado de las siguientes etiquetas: <b> y <i> define el texto en negrita y cursiva, pero <strong> y <em> significa que el texto es "importante".
