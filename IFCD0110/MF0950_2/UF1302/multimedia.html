HTML Vídeos

El HTML5 <video>elemento especifica una forma estándar para incrustar un vídeo en una página web.

Etiqueta Descripción
<video> Define un vídeo
<source> Define una fuente de vídeo o audio
<track> Define un archivo de texto en reproductores de video
<audio> Define un contenido de audio
   
   
   
   

 

<video width="320" height="240" autoplay controls preload="auto" loop poster="img/paisaje.jpg">
<source src="video/mov_bbb.mp4" type="video/mp4">
<source src="video/mov_bbb.ogg" type="video/ogg">
<track src="subtitles_en.vtt" kind="subtitles" srclang="en" label="English"></track>
</video>

HTML Audio

El HTML5 <audio> elemento especifica una forma estándar de audio para incrustar en una página web.

<audio controls>
<source src="audio/intro.mp3" type="audio/mp3">
<source src="audio/intro.ogg" type="audio/ogg">
<source src="audio/intro.wav" type="audio/wav">

</audio>

El elemento object

El <object> elemento define un objeto incrustado dentro de un documento HTML.

Se utiliza para insertar plug-ins (como applets de Java, los lectores de PDF, Flash Players) en las páginas web.

<object width="400" height="50" data="swf/bookmark.swf"></object>

El elemento embed

El <embed>elemento también define un objeto incrustado dentro de un documento HTML.

Los navegadores web han soportado el elemento <embed> durante mucho tiempo. Sin embargo, no ha sido una parte de la especificación HTML antes de HTML5.

<embed width="400" height="50" src="swf/bookmark.swf">

El elemento iframe

Marco para mostrar sitios internos o externos. Se incluyen url para que se muestren de forma autónoma. Dos ejemplos, incluyendo la source en la propia etiqueta de iframe y como enlace <a href>

<iframe width="560" height="315" src="https://www.youtube.com/embed/AO4snCT8SN4" frameborder="0" allow="accelerometer; autoplay; encrypted-media; gyroscope; picture-in-picture" allowfullscreen></iframe>
