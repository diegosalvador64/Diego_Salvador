Etiquetas que citan el texto

Las abreviaturas de marcado <abbr> pueden proporcionar información útil a los navegadores, sistemas de traducción y motores de búsqueda. Utiliza el atributo <title> para definir el significado dela abreviación

La etiqueta de marcado <cite> hace referencia a un trabajo creativo como un libro, una película, un programa de televisión o una canción, no el nombre de una persona.

La etiqueta de marcado <blockquote> define una cita larga que se representa como un elemento de bloque. No genera automáticamente comillas alrededor del contenido.

La etiqueta de marcado <q> define una cita corta en línea. Marca las citas a otros autores o documentos. Genera automáticamente comillas alrededor del contenido.

La etiqueta de marcado <dfn> define un término y debe contener el atributo <title> para la definición / explicación del término dentro.

Elementos que agrupan el texto

El <div> se utiliza a menudo como un contenedor para otros elementos HTML. El elemento <div> tiene atributos, pero no son necesarios. Los más comunes son style, class y id para dar estilos CSS al bloque de contenido.

El elemento <span> se utiliza como un contenedor para agrupar un texto en línea. El elemento <span>, también tiene atributos, pero tampoco son necesarios. Los más comunes son style, class y id para dar estilos CSS al bloque de contenido.

Etiquetas de código de computadora


  • La etiqueta <code> define un fragmento de código de ordenador
  • La etiqueta <kbd> representa comandos de entrada del usuario
  • La etiqueta <samp> representa la salida de un programa o sistema informático
  • La etiqueta <var> se utiliza como expresión matemática o variable de la programación

Etiqueta Descripción Ejemplo Utilización
<code> Define un código de programación Usar este código function(main) Usar este código <code>function(main)</code>
<kbd> Define una entrada de teclado Para guardar Ctrl + S Para guardar <kbd>Ctrl</kbd> + <kbd>S</kbd>
<samp> Define la salida de la computadora Error: página no encontrada <samp>Error: página no encontrada</samp>
<var> Define una variable aritmética Einstein escribió: E = mc2 Einstein escribió: <var>E = mc<sup>2<sup></var>

Nota: Las etiquetas de código de computadora cambian el tipo de letra simulando código informático.
