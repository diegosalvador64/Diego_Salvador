Etiquetas que definen una lista

Una tabla HTML se define con la etiqueta <table>.

Cada fila de la tabla se define con la etiqueta <tr>. Un encabezado de tabla se define con la etiqueta <th>. Por defecto, los encabezados de las tablas están en negrita y centrados. Una tabla de datos / celda se define con la etiqueta <td>.

Etiqueta Descripción
<ul> Define una lista desordenada
<ol> Define una lista ordenada
<li> Define un item de lista
<dl> Define una lista de descripción
<dt> Define un término en una lista de descripción
<dd> Describe el término en una lista de descripción

Nota: Los elementos de la lista pueden contener una nueva lista y otros elementos HTML, como imágenes, enlaces, etc.

Listas ordenadas: Ejemplos

Las listas ordenadas son casi idénticas a las listas no ordenadas, salvo que en este caso los elementos relacionados se muestran siguiendo un orden determinado. La lista ordenada se define mediante la etiqueta <ol>. Los elementos de la lista se definen mediante la etiqueta <li>, la misma que se utiliza en las listas no ordenadas. Se puede utilizar el estilo CSS list-style-type para especificar el tipo de numeración. El atributo type en la etiqueta <ol> sí se admite en HTML5

Ejemplo de listas ordenadas

    Lista type="I"

  1. Rojo
  2. Verde
  3. Azul

    Lista type="i"

  1. Rojo
  2. Verde
  3. Azul

    Lista type="a"

  1. Rojo
  2. Verde
  3. Azul

    Lista type="A"

  1. Rojo
  2. Verde
  3. Azul
  1. Rojo
  2. Verde
  3. Azul

    Lista start="10" reversed

  1. Rojo
  2. Verde
  3. Azul

Listas desordenadas

Una lista no ordenada es un conjunto de elementos relacionados entre sí para los que no se indica un orden o secuencia determinados. La etiqueta <ul> encierra todos los elementos de la lista y la etiqueta <li> cada uno de sus elementos. Se puede utilizar el estilo CSS list-style-type para especificar el tipo de viñeta. El atributo type en la etiqueta <ul> no se admite en HTML5.

Ejemplo de listas desordenadas

    Lista type= square

  • Rojo
  • Verde
  • Azul

    Lista type= disc

  • Rojo
  • Verde
  • Azul

    Lista type= none

  • Rojo
  • Verde
  • Azul

    Lista type= circle

  • Rojo
  • Verde
  • Azul
