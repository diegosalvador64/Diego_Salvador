Elementos de formulario HTML

El elemento <input>

El elemento de forma más importante es el <input>elemento.

El <input>elemento se puede mostrar en varias formas, dependiendo del type atributo.

Ejemplo:

<input name="firstname" type="text">

Datos de contacto Introduce tu nombre:

Si el type se omite, el campo de entrada obtiene el tipo por defecto: "texto".

El elemento <select>

El <select>elemento define una lista desplegable :

Coches Selecciona la marca de coche:

Los <option> elementos define una opción que puede seleccionarse.

De manera predeterminada, se selecciona el primer elemento de la lista desplegable.

Para definir una opción preseleccionada, añadir el selected atributo para la opción.

<option value="fiat" selected>Fiat</option>

Utilice el size atributo para especificar el número de valores visibles:

<select name="cars" size="3">
  <option value="volvo">Volvo</option>
  <option value="saab">Saab</option>
  <option value="fiat">Fiat</option>
  <option value="audi">Audi</option>
</select>

Utilice el multiple atributo para permitir al usuario seleccionar más de un valor:

<select name="cars" size="4" multiple>
  <option value="volvo">Volvo</option>
  <option value="saab">Saab</option>
  <option value="fiat">Fiat</option>
  <option value="audi">Audi</option>
</select>

La <textarea> Element

El <textarea> elemento define un campo de entrada de líneas múltiples ( un área de texto ):

El rows atributo especifica el número visible de líneas en un área de texto.

El cols atributo especifica la anchura visible de un área de texto.


<form action="/action_page.php">
<textarea name="message" rows="10" cols="30">The cat was playing in the garden.</textarea>

<input type="submit">
</form>

El botón <> Elemento

El <button>elemento define un hacer clic botón :

<button type="button" onclick="alert('¡Passa Tronko!')">Pulse aquí</button>

Nota: Siempre especificar el tipo de atributo para el elemento de botón. Los diferentes navegadores pueden usar diferentes tipos predeterminados para el elemento de botón.

HTML5 <lista de datos> Element

El <datalist> elemento especifica una lista de opciones predefinidas para un <input>elemento.

Los usuarios podrán ver una lista desplegable de las opciones predefinidas, ya que los datos de entrada.

El list atributo del <input> elemento, debe hacer referencia al idatributo del <datalist>elemento.

<form action="/action_page.php">
  <input list="browsers">
  <datalist id="browsers">
    <option value="Internet Explorer">
    <option value="Firefox">
    <option value="Chrome">
    <option value="Opera">
    <option value="Safari">
  </datalist> 
</form>
