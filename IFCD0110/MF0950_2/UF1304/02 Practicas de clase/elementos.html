El elemento <input>

El elemento de forma más importante es el <input>elemento.

El <input>elemento se puede mostrar en varias formas, dependiendo del type atributo.

Ejemplo:

<input name="firstname" type="text">

El <select>elemento define una lista desplegable :

Los <option> elementos define una opción que puede seleccionarse.

De manera predeterminada, se selecciona el primer elemento de la lista desplegable.

Para definir una opción preseleccionada, añadir el selected atributo para la opción.

<option value="fiat" selected>Fiat</option>

Utilice el size atributo para especificar el número de valores visibles:

<select name="cars" size="3">
  <option value="volvo">Volvo</option>
  <option value="saab">Saab</option>
  <option value="fiat">Fiat</option>
  <option value="audi">Audi</option>
</select>

Etiqueta <label>

La etiqueta <label> define una etiqueta para un botón <>, <input>, <metro>, <salida>, <progreso>, <seleccionar> o elemento <textarea>.

El elemento <label> no hace nada especial para el usuario, pero mejora la facilidad de uso para los usuarios, ya que si el usuario hace clic en el texto dentro del elemento <label>, se alterna el control.

El atributo de la etiqueta <label> debería ser igual al atributo id del elemento relacionado para unirse juntos.

<form action="/action_page.php">
<label for="male">Male</label>
<input type="radio" name="gender" id="male" value="male"><br>
<label for="female">Female</label>
<input type="radio" name="gender" id="female" value="female"><br>
<label for="other">Other</label>
<input type="radio" name="gender" id="other" value="other"><br><br>
<input type="submit" value="Submit">
</form>

Elemento <textarea>

El <textarea> elemento define un campo de entrada de líneas múltiples ( un área de texto ):

El rows atributo especifica el número visible de líneas en un área de texto.

El cols atributo especifica la anchura visible de un área de texto.


<form action="/action_page.php">
<textarea name="message" rows="10" cols="30">The cat was playing in the garden.</textarea>
<br>
<input type="submit">
</form>

El Elemento <button>

El <button>elemento define un hacer clic botón :

<button type="button" onclick="alert('¡Passa Tronko!')">Pulse aquí</button>

Nota: Siempre especificar el tipo de atributo para el elemento de botón. Los diferentes navegadores pueden usar diferentes tipos predeterminados para el elemento de botón.

HTML5 Elemento <lista de datos>

El <datalist> elemento especifica una lista de opciones predefinidas para un <input>elemento. Los usuarios podrán ver una lista desplegable de las opciones predefinidas, ya que los datos de entrada. El list atributo del <input> elemento, debe hacer referencia al id atributo del elemento <datalist>. Ejemplos con varios atributos de <form>. ESTA ETIQUETA HTML5 NO ES COMPATIBLE CON SAFARI.

<form action="/action_page.php" method="get" accept-charset="UTF-8" autocomplete="on" enctype="multipart/form-data" name="formulario5" novalidate target="_blank">
<fieldset>
<legend>Elementos datalist</legend>
<input list="navegadores">
<datalist id="navegadores">
<option value="Internet Explorer"></option>
<option value="Firefox"></option>
<option value="Chrome"></option>
<option value="Opera"></option>
<option value="Safari"></option>
</datalist>
<input type="submit" value="Enviar">
</fieldset>
</form>

Elementos de formulario HTML

 

Tag Description
<form> Defines an HTML form for user input
<input> Defines an input control
<textarea> Defines a multiline input control (text area)
<label> Defines a label for an <input> element
<fieldset> Groups related elements in a form
<legend> Defines a caption for a <fieldset> element
<select> Defines a drop-down list
<optgroup> Defines a group of related options in a drop-down list
<option> Defines an option in a drop-down list
<button> Defines a clickable button
<datalist> Specifies a list of pre-defined options for input controls
<output> Defines the result of a calculation
