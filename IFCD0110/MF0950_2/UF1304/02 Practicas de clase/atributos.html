Atributo autocomplete

Especifica si un campo de formulario o de entrada debe tener autocompletar encendido o apagado.

Cuando autocompletar está activado, el navegador completa automáticamente los valores de entrada basado en valores que el usuario ha introducido antes.

El atributo autocomplete trabaja con <form> y los siguientes <input> types: text, search, url, tel, email, password, datepickers, range, and color.

Atributo autocomplete Nombre

Apellido

El atributo required trabaja con los siguientes tipos de entrada: ext, search, url, tel, email, password, date pickers, number, checkbox, radio, y file.

Atributo step

Especifica los intervalos de números legales para un <input>elemento. El atributo step trabaja con los elementos input number, range, date, datetime-local, month, time y week.

Ejemplo: Si el paso = "3", números legales podrían ser -3, 0, 3, 6, etc.
