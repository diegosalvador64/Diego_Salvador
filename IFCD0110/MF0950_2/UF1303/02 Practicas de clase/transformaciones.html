CSS Transformaciones 2D

Las transformaciones CSS permiten trasladar, rotar, escalar y sesgar elementos. Una transformación es un efecto que permite un cambio de elemento de forma, tamaño y posición. CSS admite transformaciones 2D y 3D.

Transformaciones CSS 2D

Los métodos de transformación 2D son:

  • translate()
  • rotate()
  • scale()
  • skewX()
  • skewY()
  • matrix()

El Método translate ()

El método translate() mueve un elemento desde su posición actual (de acuerdo con los parámetros dados para el eje X y el eje Y).

Original
Translate

El Método rotate ()

El método rotate() hace girar un sentido horario elemento o antihorario según un grado dado.

Original
Rotate

Método scale ()

El método scale() aumenta o disminuye el tamaño de un elemento (de acuerdo con los parámetros dados para la anchura y altura).

Original
Scale

Método skew ()

El método skew() sesga o inclina un elemento a lo largo de la X y el eje Y por los ángulos dados. Es la fusión de dos métodos:

  • El método skewX() sesga un elemento a lo largo del eje X por el ángulo dado.
  • El método skewY() sesga un elemento a lo largo del eje Y por el ángulo dado.
Original
Skew

Método matrix ()

El método matrix() combina todo el 2D métodos de transformada en una sola.

El método matrix () toma seis parámetros, que contiene funciones matemáticas, que le permite rotar, escala, mover e inclinar elementos.
