¿Qué es CSS?

  • CSS significa Cascading Style Sheets (Hojas de Estilos en Casacada)
  • CSS describe cómo los elementos HTML se van a mostrar en la pantalla, papel, o en otros medios
  • CSS ahorra mucho trabajo . Se puede controlar el diseño de varias páginas web a la vez
  • Estilo externas se almacenan en archivos CSS

Estructura básica de un documento HTML

Código HTML

<!DOCTYPE html>
<html lang="es-ES">

<!-- cabeza del documento HTML-->
<head>
<title>Título de página</title>
<meta charset="utf-8">
</head>

<!-- cuerpo del documento HTML -->
<body>
<h1>Encabezado de página</h1>
<p>Párrafo de texto</p>
</body>

</html>

Explicación de las etiquetas básicas HTML

  1. La declacración<!DOCTYPE html> define el tipo de documento.
  2. El elemento <html> define la estructura raíz del documento HTML.
  3. El atributo lang="es-ES" especifica el idioma y país texto del documento HTML.
  4. La sección <head> define la información acerca del documento HTML.
  5. La etiqueta <title> define el título del documento HTML.
  6. La etiqueta <meta charset="utf-8"> define la codificación de caracteres.
  7. La sección<body> define el cuerpo del documento HTML.
  8. La etiqueta <h1> define un encabezado de texto.
  9. La etiqueta <p> define un párrafo de texto.
  10. La etiqueta <!-- ... --> inserta comentarios en el código HTML.

Etiquetas HTML

Las etiquetas HTML son los nombres de los elementos entre paréntesis angulares.

<elemento atributo="valor"> el contenido va aquí ... </elemento>

Etiquetas HTML
Se escriben por pares como <p> y </p>, pero también solas como <br>.
Etiqueta inicio
La primera etiqueta en un par es la etiqueta de inicio, y la seguna la etiqueta de fin.
Etiqueta fin
Posee una barra inclinada que se inserta antes del nombre de la etiqueta </html>.

Nota: La etiqueta de inicio también se llama etiqueta de apertura y la etiqueta final etiqueta de cierre.

Navegadores web

El propósito de un navegador web es leer los documentos HTML y mostrarlos.
El navegador no muestra las etiquetas HTML, pero las utiliza para determinar como mostrar el documento.

Estructura de una página HTML

A continuación se muestra una visualización de una estructura de la página HTML:

<html>
<head>
<link rel="stylesheet" href="css/estilos.css" type="text/css">
<style>h1 {font-size:14px; color:Blue} </style>
</head>
<body>
<h1>Este es un encabezado de texto</h1>
<p>Esto es un párrafo de texto</p>
</body>
</html>

Nota: Sólo el contenido dentro de la sección <body> (el área blanca arriba) se muestra a través de un navegador web.
