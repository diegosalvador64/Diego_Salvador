Las transiciones CSS

Las transiciones CSS le permiten cambiar los valores de propiedad sin problemas (de un valor a otro), durante un período determinado.

Propiedades CSS Transición

La siguiente tabla muestra todas las propiedades de transición CSS:

Property Description
transition A shorthand property for setting the four transition properties into a single property
transition-delay Especifica el retardo de tiempo (en segundos) para el efecto de transición
transition-duration Especifica duración en segundos (s) o milisegundos (ms) pare el efecto de transición
transition-property Specifies the name of the CSS property the transition effect is for
transition-timing-function Specifies the speed curve of the transition effect

Especificar la curva de velocidad de la Transición

La propiedad transition-timing-function especifica la curva de velocidad del efecto de transición.

La propiedad-timing-función de transición puede tener los siguientes valores:

  • ease - especifica un efecto de transición con un comienzo lento, luego rápido, y luego terminar lentamente (esto es por defecto)
  • linear - especifica un efecto de transición con la misma velocidad de principio a fin
  • ease-in - especifica un efecto de transición con un comienzo lento
  • ease-out - especifica un efecto de transición con un extremo lento
  • ease-in-out - especifica un efecto de transición con un lento comienzo y fin
  • cubic-bezier(n,n,n,n) - le permite definir sus propios valores de una función cúbica-Bezier

linear

ease

ease-in

ease-out

ease-in-out

Retrasar el efecto de transición

La propiedad transition-delay especifica un retardo (en segundos) para el efecto de transición.

Ejemplo de uso

linear delay 0.5s

ease delay 1s

ease-in delay 1.5s

ease-out delay 2s

ease-in-out delay 2s

Propiedad duración del efecto de transición

La propiedad transition-duration especifica el número de segundos (s) o en milisegundos (ms) un efecto de transición tarda en completar.

Ejemplo de uso

linear delay 0.5s duration 2s

ease delay 1s duration 2s

ease-in delay 1.5s duration 2s

ease-out delay 2s duration 2s

ease-in-out delay 2s duration 2s

Propiedad transition-property

La propiedad transition-property especifica el nombre de la propiedad CSS el efecto de transición es para (el efecto de transición se iniciará cuando cambia la propiedad CSS especificados).

Nota: Siempre especifique la transición de la duración de la propiedad, de lo contrario la duración es 0, y la transición no tendrá ningún efecto.

Propiedad transition-property

Propiedad transition

La propiedad transition es una propiedad abreviada para:

  • transition-property
  • transition-duration
  • transition-timing-function
  • transition-delay
