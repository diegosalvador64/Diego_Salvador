Métodos Fechas (Date)

El concepto de fecha

Por defecto, JavaScript utilizará la zona horaria del navegador y mostrar una fecha como una cadena de texto completo:

Mar Mar 19 de 2019 12:50:06 GMT + 0100 (Hora Estándar de Europa central)

Los objetos de Fecha se crean con el constructor new Date(). Crea un nuevo objeto Date con la fecha y hora actual.

Métodos Fechas

Método Descripción Síntaxis  
new Date() Crea un nuevo objeto de tipo Date con la fecha y hora actual. new Date()
getFullYear() Método que devuelve el año de una fecha como un número de cuatro dígitos getFullYear()
getMonth() Método que devuelve el mes de una fecha como un número (0-11) getMonth()
getDate() Método que devuelve el día de una fecha como un número (1-31) getDate()
getHours() Método que devuelve las horas de una fecha como un número (0-23) getHours()
getMinutes() Método que devuelve los minutos de una fecha como un número (0-59) getMinutes()
getSeconds() Método devuelve los segundos de una fecha como un número (0-59) getSeconds()
getMilliseconds() Método que devuelve los milisegundos de una fecha como un número (0-999): getMilliseconds()
getTime() Método que devuelve el número de milisegundos desde 1 de enero de 1970: getTime()
getDay() Método que devuelve el día de la semana de una fecha como un número (0-6) getDay()
